# Infinite Math Test
Infinite Math Test is a lightweight training app designed to help you sharpen your mental arithmetic skills through an endless stream of simple, fast‑paced math problems. Whether you're practicing for school, improving your reaction speed, or just want a quick brain workout, the app provides a clean and distraction‑free environment to keep your mind active.

# Features
- Infinite stream of randomly generated math problems
- Clean and minimalistic interface (Qt + QML)
- Instant feedback on each answer

# VirusTotal Notice
Some antivirus engines may flag the executable as suspicious due to it being a small, unsigned Qt application.
These detections are false positives - the full source code is available in this repository, and the app contains no harmful functionality.

# How to Run
- Download the latest release from the Releases section
- Unzip the archive
- Run "Infinite Math Test.exe"
